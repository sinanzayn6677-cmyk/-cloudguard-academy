🛡️ CloudGuard Academy Workflow

Complete user journey and system architecture

1
👤

User Registration

New users create an account by entering email and password. Password is hashed with bcryptjs for security.

⬇️
2
🔑

JWT Authentication

System generates a JWT token (valid for 24 hours) that authenticates user requests and maintains secure sessions.

⬇️
3
📊

Dashboard Access

User lands on dashboard showing available labs, total points, completed labs, and current ranking on leaderboard.

⬇️
4
🎯

Choose Lab

User selects a security lab to learn from: SQL Injection, XSS, CSRF, or IDOR. Each lab has two modes: Vulnerable & Secure.

⬇️
5
💻

Practice & Learn

In Vulnerable mode, users exploit the vulnerability. In Secure mode, they see how to fix it. Side-by-side code comparison included.

⬇️
6
🚩

Submit Flag

Upon successful exploitation, user receives a flag. They submit it to verify their exploit and earn points.

⬇️
7
⭐

Earn Points & Rank

System validates the flag and awards points (100-150 per lab). User's ranking updates in real-time on the leaderboard.

⬇️
8
🏆

Check Leaderboard

View global rankings and compete with other security learners. Track progress and strive to complete all 450 points.

🔬 Individual Lab Workflows

🔐 SQL Injection Lab

1️⃣ Read vulnerability explanation
2️⃣ Try vulnerable login form
3️⃣ Bypass with: ' OR '1'='1
4️⃣ Review secure parameterized query
5️⃣ Submit flag: FLAG{SQL_INJECTION_MASTER}
✅ Earn 100 points

🎯 XSS Lab

1️⃣ Understand XSS vectors
2️⃣ Try vulnerable forum post
3️⃣ Inject: <img onerror="alert('XSS')">
4️⃣ Compare with HTML entity encoding
5️⃣ Submit flag: FLAG{XSS_SCRIPT_INJECTOR}
✅ Earn 100 points

⚔️ CSRF Lab

1️⃣ Learn CSRF attack vectors
2️⃣ Vulnerable: No token validation
3️⃣ Transfer funds without token
4️⃣ Review token-based protection
5️⃣ Submit flag: FLAG{CSRF_FORGED_REQUEST}
✅ Earn 100 points

🔓 IDOR Lab

1️⃣ Understand object reference
2️⃣ Access DOC-001 (your document)
3️⃣ Change to DOC-002 (other user)
4️⃣ Review authorization checks
5️⃣ Submit flag: FLAG{IDOR_ACCESS_VIOLATION}
✅ Earn 150 points

CORS Protection

Rate Limiting

Input Validation

HTTP security headers prevent common attacks
